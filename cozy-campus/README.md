# Cozy Campus Engine

Protótipo de engine em Three.js para ambientes interativos de um cozy game/RPG de navegador. O projeto prioriza dioramas vivos, interação por clique e uma arquitetura que mantenha gameplay, renderização e interface separados.

## Executar

```bash
npm install
npm run dev
```

## Documentação

O ponto de entrada é [docs/README.md](docs/README.md). A documentação é mantida junto com cada sistema para registrar decisões e facilitar o aprendizado.
